# Habbify
## The Idea
A small gamified habit tracking prototype.

### Functions
- Create new habits
  - Everyday habits
  - X times a week habits
  - (X times a month habits)
  - (X times a year habits)
- Track habit
- Earn EXP
- Level Up
- (Compete against others in season leaderboards)
